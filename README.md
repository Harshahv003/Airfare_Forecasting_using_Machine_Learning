## рџ“Њ Project Overview

**FlightFare AI** is a production-ready airfare price prediction system built with machine learning. It predicts Indian domestic flight ticket prices based on airline, route, travel date, departure time, and number of stops.

The system was trained on **10,682 real flight records** and achieves an **RВІ score of 0.84** using a Gradient Boosting model вЂ” meaning it explains 84% of the variance in airfare pricing.

## вљ™пёЏ Installation & Setup

### Prerequisites
- Python 3.11+
- pip

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/airfare-prediction.git
cd airfare-prediction
```

### 2. Create virtual environment

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add the dataset

Place `Data_Train.xlsx` in the `dataset/` folder.

### 5. Train the model

```bash
python model/train_model.py
```

This will:
- Clean and engineer features
- Train Linear Regression, Random Forest, and Gradient Boosting
- Run hyperparameter tuning
- Save the best model to `model/flight_price_model.pkl`
- Generate 8 EDA + evaluation plots in `static/plots/`

### 6. Run the app

```bash
python app.py
```

Open: **http://127.0.0.1:5000**

## рџ§  ML Pipeline Details

### Dataset
- **10,682** flight records
- **11 features** в†’ engineered to **13 model features**
- Date range: MarchвЂ“June 2019 (Indian domestic flights)

### Feature Engineering

| Raw Feature | Engineered Feature |
|---|---|
| `Date_of_Journey` | `Journey_Day`, `Journey_Month` |
| `Dep_Time` | `Dep_Hour`, `Dep_Minute`, `TimeSlot_enc` |
| `Arrival_Time` | `Arrival_Hour`, `Arrival_Minute` |
| `Duration` | `Duration_Minutes` |
| `Total_Stops` | `Stops` (integer 0вЂ“4) |
| `Route` | `Route_Hops` |
| `Airline` | `Airline_enc` (LabelEncoder) |
| `Source` | `Source_enc` |
| `Destination` | `Destination_enc` |

### Model Comparison

| Model | RВІ | MAE | RMSE |
|---|---|---|---|
| Linear Regression (baseline) | 0.4381 | в‚№2,448 | в‚№3,481 |
| Random Forest (default) | 0.7963 | в‚№1,186 | в‚№2,096 |
| Gradient Boosting | **0.8396** | **в‚№1,213** | **в‚№1,860** |
| Random Forest (tuned) | 0.8217 | в‚№1,219 | в‚№1,960 |

### Key Findings from EDA
- **Stops** have the strongest correlation with price (r=0.60)
- **Duration** is second strongest (r=0.51)
- **Jet Airways Business** is 3Г— more expensive than economy average
- **Non-stop flights** average в‚№5,025 vs в‚№17,686 for 4-stop flights
- Prices peak in **MayвЂ“June** (summer travel season)
